Propriedades

  • align-content - Especifica o alinhamento entre as linhas dentro de um contêiner flexível quando os itens não ocupam todo o espaço disponível.

  • align-items - Especifica o alinhamento dos itens dentro de um contêiner flexível.

  • align-self - Especifica o alinhamento dos itens selecionados dentro de um contêiner flexível.

  • flex-basis - Especifica o comprimento inicial de um item flexível.

  • flex-direction - Especifica a direção dos itens flexíveis.

  • flex-grow - Especifica quanto o item crescerá em relação ao restante.

  • flex-shrink - Especifica como o item diminuirá em relação ao restante.

  • flex-wrap - Especifica se os itens flexíveis devem ser empacotados ou não.

  • order - Define a ordem do item flexível, em relação ao resto.
