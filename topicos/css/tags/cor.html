Propriedades

  • accent-color - Especifica uma cor de destaque para os controles da interface do usuário.

  • background - Uma propriedade para todas as propriedades background.

  • background-blend-mode - Especifica o modo de mesclagem de cada camada de fundo (cor/imagem).

  • background-clip - Define até onde o fundo (cor ou imagem) deve se estender dentro de um elemento.

  • background-color - Especifica a cor de fundo de um elemento.

  • border - Uma propriedade abreviada para largura da borda, estilo da borda e cor da borda.

  • border-bottom - Uma propriedade abreviada para largura inferior da borda. estilo inferior da borda e cor inferior da borda.

  • border-bottom-color - Define a cor da borda inferior.

  • border-color - Define a cor das quatro bordas.

  • border-left-color - Define a cor da borda esquerda.

  • border-right-color - Define a cor da borda direita.

  • border-top-color - Define a cor da borda superior.

  • caret-color - Especifica a cor do cursor (acento circunflexo) em entradas, áreas de texto ou qualquer elemento que seja editável.

  • color - Defina a cor do texto.

  • column-rule-color - Especifica a cor da regra entre colunas.

  • filter - Define efeitos (por exemplo, desfoque ou mudança de cor) em um elemento antes que o elemento seja exibido.

  • outline - Uma propriedade abreviada para as propriedades de largura do contorno, estilo de contorno e cor de contorno.

  • outline-color - Define a cor de um contorno.

  • text-decoration-color - Especifica a cor da decoração de texto.
