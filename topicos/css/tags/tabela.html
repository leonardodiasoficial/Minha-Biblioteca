Tags

  • !-- - Define um comentário no HTML.

  • !DOCTYPE - Declaração para informar ao navegador qual a versão do HTML utilizada no arquivo.

  • html - Define um documento HTML.

  • head - Contém as informações gerais (metadados) do documento.

  • title - Define o título da página.

  • base - Utilizada para específicar a URL base do seu site, esta tag faz a linkagem com links internos mais limpas.

  • meta - Está tag do meta da página. Pode ser útil para mencionar o autor da página, palavras-chave, datas de publicação, etc.

  • link - Utilizada para linkar com links externos à página.

  • style - Utilizada como uma alternativa à uma stylesheet externa, ou para complementá-la. Inclui as informações de aparência da página.

  • script - Utilizada para adicionar códigos snippet, tipicamente em JavaScript, para tornar a página dinâmica. Também usada para linkar à um script externo.
