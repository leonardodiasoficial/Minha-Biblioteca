Tags e Atributos

  • strong - Deixa o texto em negrito. Usado para enfatizar o conteúdo.

  • b - Alternativa à tag acima, também deixa o texto em negrito

  • em - Outra tag para enfatizar, mas esta mostra o texto em itálico.

  • i - Também mostra o texto em itálico, mas não enfatiza como a tag acima.

  • cite - Tag para citar o autor de uma frase.

  • del - Texto pré-formatado "monoespaçado" apresentado com espaço em branco dentro do elemento intacto.

  • ins - Denota um texto que foi inserido na página.

  • blockquote - Citações são geralmente colocadas com esta tag. Ela é utilizada em conjunto com a tag cite.

  • q - Parecida com a tag acima, mas para citações menores.

  • abbr - Denota abreviações, juntamente com a forma completa.

  • address - Tag para especificar as informações de contato do autor.

  • dfn - Tag dedicada para definições.

  • code - Utilizada para mostrar códigos snippet dentro de um parágrafo.

  • sub - Tag para escrita de um subscript (um texto de fonte menor logo abaixo da fonte principal).

  • sup - Similar com a tag anterior, mas para superscript (texto acima).

  • small - Reduz o tamanho do texto. No HTML5 geralmente se refere à informações redundantes ou inválidas.

  • bdo - Usada para substituir a direção do texto atual.

  • data - Usada para adicionar uma tradução legível por máquina de um determinado conteúdo.

  • s - Especifica o texto que não é mais correto, preciso ou relevante. O texto será exibido com uma linha através dele.

  • samp - Usada para definir a saída de amostra de um programa de computador. O conteúdo interno é exibido na fonte padrão do navegador

  • u - O conteúdo interno é exibido com um sublinhado, representando um texto não articulado e com estilo diferente do texto normal.

  • output - Usada para representar o resultado de um cálculo (como um realizado por um script).

  • var - O conteúdo interno é exibido em itálico, usada para definir uma variável na programação ou em uma expressão matemática.

  • kbd - Usada para definir a entrada do teclado. O conteúdo é exibido na fonte padrão do navegador
