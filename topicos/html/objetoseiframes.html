Tags e Atributos

  • object - Esta tag é utilizada para incorporar tipos de mídia adicionais à página. Pode ser áudio, vídeo, documento (pdf) e etc.

  • iframe - Um bloco de conteúdo em linha, utilizado como um contâiner flexível para arquivos de mídia. Ele flutua dentro da página, significando que é colocado relativo à outros itens da página.

  • canvas - Usada para desenhar gráficos, em tempo real, por meio de scripts.

  • param - Usada para definir parâmetros para um elemento object.
