<canvas>

Descrição

Usada para desenhar gráficos, em tempo real, por meio de scripts (geralmente JavaScript).

Transparente e é apenas um container para gráficos, você deve usar um script para realmente desenhar os gráficos.

Qualquer texto dentro do <canvas> será exibido em navegadores com JavaScript desabilitado e em navegadores que não suportam <canvas>.

Atributos

height - Especifica a altura da tela. O valor padrão é 150.

width - Especifica a largura da tela O valor padrão é 300.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<canvas id="myCanvas">
    Your browser does not support the canvas tag.
</canvas>

Configuração Padrão de CSS

                
canvas {
    height: 150px;
    width: 300px;
}

Compatibilidade dos Navegadores

  • Chrome = ✔️ (4.0)

  • Edge = ✔️ (9.0)

  • Firefox = ✔️ (2.0)

  • Safari = ✔️ (3.1)

  • Opera = ✔️ (9.0)
