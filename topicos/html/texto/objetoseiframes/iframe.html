<iframe>

Descrição

Especifica um quadro embutido.

Um quadro embutido é usado para incorporar outro documento no documento HTML atual.

Notas

Use CSS para estilizar <iframe>

É uma boa prática sempre incluir um atributo de título para o arquivo <iframe>. Isso é usado por leitores de tela para ler qual <iframe> é o conteúdo.

Atributos

allow - Especifica uma política de recursos para o <iframe>.

allowfullscreen - Defina como verdadeiro se o <iframe> puder ativar o modo de tela cheia chamando o método requestFullscreen().

allowpaymentrequest - Defina como true se um <iframe> de origem cruzada deve ter permissão para invocar a API de solicitação de pagamento.

height - Especifica a altura de um <iframe>. Altura padrão é 150 pixels.

loading - Especifica se um navegador deve carregar um iframe imediatamente ou adiar o carregamento de iframes até que algumas condições sejam atendidas.

name - Especifica o nome de um <iframe>.

referrerpolicy - Especifica quais informações do referenciador enviar ao buscar o iframe.

sandbox - Habilita um conjunto extra de restrições para o conteúdo em um <iframe>.

src - Especifica o endereço do documento a ser inserido no <iframe>.

srcdoc - Especifica o conteúdo HTML da página para mostrar no <iframe>.

width - Especifica a largura de um <iframe>. A largura padrão é de 300 pixels.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<iframe src="https://www.w3schools.com" title="W3Schools Free Online Web Tutorials"></iframe>

Configuração Padrão de CSS

                
iframe:focus {
    outline: none;
}
    
iframe[seamless] {
    display: block;
}
