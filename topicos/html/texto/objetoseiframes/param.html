<param>

Descrição

usada para definir parâmetros para um elemento <object>.

Atributos

name - Especifica o nome de um parâmetro.

value - Especifica o valor do parâmetro.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<object data="horse.wav">
    <param name="autoplay" value="true">
</object>

Configuração Padrão de CSS

                
param {
    display: none;
}
