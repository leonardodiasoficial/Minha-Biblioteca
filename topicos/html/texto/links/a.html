<a>

Descrição

Define um hiperlink, que é usado para vincular de uma página a outra.

O atributo mais importante do <a> é o atributo href, que indica o destino do link.

Notas

Se a <a> não tiver nenhum hrefatributo, ela é apenas um espaço reservado para um hiperlink.

Uma página vinculada normalmente é exibida na janela do navegador atual, a menos que você especifique outro destino.

Atributos

download - Especifica que o destino será baixado quando um usuário clicar no hiperlink.

href - Especifica o URL da página para a qual o link vai.

hreflang - Especifica o idioma do documento vinculado.

media - Especifica para qual mídia/dispositivo o documento vinculado é otimizado.

ping - Especifica uma lista separada por espaço de URLs para a qual, quando o link for seguido, as solicitações de postagem com o ping do corpo serão enviadas pelo navegador (em segundo plano). Normalmente usado para rastreamento.

referrerpolicy - Especifica quais informações do referenciador enviar com o link.

rel - Especifica a relação entre o documento atual e o documento vinculado.

target - Especifica onde abrir o documento vinculado.

type - Especifica o tipo de mídia do documento vinculado.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<a href="https://www.w3schools.com">Visit W3Schools.com!</a>

Configuração Padrão de CSS

                
a:link, a:visited {
    color: (internal value);
    text-decoration: underline;
    cursor: auto;
}
    
a:link:active, a:visited:active {
    color: (internal value);
}
