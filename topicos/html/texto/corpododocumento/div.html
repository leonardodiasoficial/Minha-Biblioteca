<div>

Descrição

Define uma divisão ou uma sessão em um documento HTML.

Usada como um contêiner para elementos HTML - que são estilizados com CSS ou manipulados com JavaScript.

Facilmente estilizada usando o atributo class ou id.

Qualquer tipo de conteúdo pode ser colocado dentro da tag!

Notas

Por padrão, os navegadores sempre colocam uma quebra de linha antes e depois do <div>.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<div class="myDiv">
    <h2>This is a heading in a div element</h2>
    <p>This is some text in a div element.</p>
</div>

Configuração Padrão de CSS

                
div {
    display: block;
    }
