<footer>

Descrição

Define um rodapé para um documento ou seção.

Um <footer> normalmente contém:

- Informações de autoria;

- Informações sobre direitos autorais;

- Informações de contato;

- Mapa do site;

- Voltar aos links principais;

- Documentos relacionados.

Notas

As informações de contato dentro de um <footer> devem estar dentro de uma tag <address>.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos artibutos de evento em HTML.

Exemplo

                
<footer>
    <p>Author: Hege Refsnes</p>
    <p><a href="mailto:hege@example.com">hege@example.com</a></p>
</footer>

Configuração Padrão de CSS

                
footer {
    display: block;
    }
