<template>

Descrição

Usada como um contêiner para manter algum conteúdo HTML oculto do usuário quando a página é carregada.

O conteúdo interno pode ser renderizado posteriormente com um JavaScript.

Você pode usar a tag se tiver algum código HTML que deseja usar várias vezes, mas não até que você solicite. Para fazer isso sem a tag, você precisa criar o código HTML com JavaScript para evitar que o navegador renderize o código.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Exemplo

                
<template>
    <h2>Flower</h2>
    <img src="img_white_flower.jpg" width="214" height="204">
</template>
