<dialog>

Descrição

Define uma caixa de diálogo ou subjanela.

Facilita a criação de caixas de diálogo pop-up e modais em uma página da web.

Atributos

open - Especifica que o elemento de diálogo está ativo e que o usuário pode interagir com ele

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<dialog open>This is an open dialog window</dialog>

Configuração Padrão de CSS

                
Não sabemos

Compatibilidade dos Navegadores

  • Chrome = ✔️ (37.0)

  • Edge = ✔️ (79.0)

  • Firefox = ✔️ (53.0)*

  • Safari = ❌

  • Opera = ✔️ (24.0)

* Não suportado por padrão, mas pode ser habilitado em about:config (defina dom.dialog_element.enabled como true).
