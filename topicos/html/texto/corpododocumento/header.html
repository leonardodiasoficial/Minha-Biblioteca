<header>

Descrição

Representa um contêiner para conteúdo introdutório ou um conjunto de links de navegação.

Um <header> normalmente contém:

- Um ou mais elementos de cabeçalho (<h1> - <h6>);

- Logotipo ou ícone;

- Informações de autoria.

Notas

Você pode ter vários <header> elementos em um documento HTML. No entanto, <header> não pode ser colocado dentro de um <footer>, <address> ou outro <header> elemento.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<header>
    <h1>A heading here</h1>
    <p>Posted by John Doe</p>
    <p>Some additional information here</p>
</header>

Configuração Padrão de CSS

                
header {
    display: block;
    }
