<nav>

Descrição

Define um conjunto de links de navegação.

Observe que nem todos os links de um documento devem estar dentro de um <nav>. O <nav> destina-se apenas aos principais blocos de links de navegação.

Navegadores, como leitores de tela para usuários desativados, podem usar esse elemento para determinar se devem omitir a renderização inicial desse conteúdo.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<nav>
    <a href="/html/">HTML</a> 
    <a href="/css/">CSS</a> 
    <a href="/js/">JavaScript</a> 
    <a href="/python/">Python</a>
</nav>

Configuração Padrão de CSS

                
nav {
    display: block;
    }
