<article>

Descrição

Especifica conteúdo independente e independente.

Um artigo deve fazer sentido por si só e deve ser possível distribuí-lo independentemente do resto do site.

Fontes potenciais para o elemento:

- Postagem no fórum;

- Postagem no blog;

- Notícia.

Notas

o <article> não é renderizado como algo especial em um navegador. No entanto, você pode usar CSS para estilizar o <article>.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<article>
    <h2>Google Chrome</h2>
    <p>Google Chrome is a web browser</p>
</article>

Configuração Padrão de CSS

                
article {
    display: block;
    }
