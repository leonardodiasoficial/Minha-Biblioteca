<th>

Descrição

Define uma célula de cabeçalho em uma tabela HTML.

Uma tabela HTML tem dois tipos de células:

Células de cabeçalho - contém informações de cabeçalho (criadas com o <th>);

Células de dados - contém dados (criados com o elemento <td>).

Notas

O texto nos <th> está em negrito e centralizado por padrão.

O texto nos elementos <td> são regulares e alinhados à esquerda por padrão.

Atributos

abbr - Especifica uma versão abreviada do conteúdo em uma célula de cabeçalho.

colspan - Especifica o número de colunas que uma célula de cabeçalho deve abranger.

headers - Especifica uma ou mais células de cabeçalho às quais uma célula está relacionada.

rowspan - Especifica o número de linhas que uma célula de cabeçalho deve abranger.

scope - Especifica se uma célula de cabeçalho é um cabeçalho para uma coluna, linha ou grupo de colunas ou linhas.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<table>
    <tr>
        <th>Month</th>
        <th>Savings</th>
    </tr>
    <tr>
        <td>January</td>
        <td>$100</td>
    </tr>
    <tr>
        <td>February</td>
        <td>$80</td>
    </tr>
</table>

Configuração Padrão de CSS

                
th {
    display: table-cell;
    vertical-align: inherit;
    font-weight: bold;
    text-align: center;
}
