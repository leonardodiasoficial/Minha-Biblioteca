<td>

Descrição

Define uma célula de dados padrão em uma tabela HTML.

Uma tabela HTML tem dois tipos de células:

Células de cabeçalho - contém informações de cabeçalho (criadas com o elemento <th>);

Células de dados - contém dados (criados com o <td>).

Notas

O texto nos <td> é regular e alinhado à esquerda por padrão.

O texto nos elementos <th> está em negrito e centralizado por padrão.

Atributos

colspan - Especifica o número de colunas que uma célula deve abranger.

headers - Especifica uma ou mais células de cabeçalho às quais uma célula está relacionada.

rowspan - Define o número de linhas que uma célula deve abranger.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<table>
    <tr>
        <td>Cell A</td>
        <td>Cell B</td>
    </tr>
    <tr>
        <td>Cell C</td>
        <td>Cell D</td>
    </tr>
</table>

Configuração Padrão de CSS

                
td {
    display: table-cell;
    vertical-align: inherit;
}
