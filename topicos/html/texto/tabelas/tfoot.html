<tfoot>

Descrição

Usada para agrupar o conteúdo do rodapé em uma tabela HTML.

Usado em conjunto com os elementos <thead> e <tbody> para especificar cada parte de uma tabela (rodapé, cabeçalho, corpo).

Os navegadores podem usar esses elementos para permitir a rolagem do corpo da tabela independentemente do cabeçalho e do rodapé. Além disso, ao imprimir uma tabela grande que abrange várias páginas, esses elementos podem permitir que o cabeçalho e o rodapé da tabela sejam impressos na parte superior e inferior de cada página.

Notas

O <tfoot> deve ter uma ou mais tags <tr> dentro.

A <tfoot> deve ser usada no seguinte contexto: Como filho de um elemento <table>, após qualquer elemento <caption>, <colgroup>, <thead> e <tbody>.

Os elementos <thead>, <tbody> e <tfoot> não afetarão o layout da tabela por padrão. No entanto, você pode usar CSS para estilizar esses elementos.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<table>
    <thead>
    </thead>
    <tbody>
    </tbody>
    <tfoot>
        <tr>
        <td>Sum</td>
        <td>$180</td>
        </tr>
    </tfoot>
</table>

Configuração Padrão de CSS

                
tfoot {
    display: table-footer-group;
    vertical-align: middle;
    border-color: inherit;
}
