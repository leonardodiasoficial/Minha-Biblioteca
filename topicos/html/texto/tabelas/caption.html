<caption>

Descrição

Define uma legenda de tabela.

Deve ser inserida imediatamente após a tag <table>.

Notas

Por padrão, uma legenda de tabela será alinhada ao centro acima de uma tabela. No entanto, as propriedades CSS text-align e caption-side podem ser usadas para alinhar e colocar a legenda.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<tabl>
    <captio>Monthly savings</captio>
    <t>
        <th>Month</t>
    </t>
    <t>
        <t>January</t>
    </t>
</tabl>

Configuração Padrão de CSS

                
caption {
    display: table-caption;
    text-align: center;
}
