<tr>

Descrição

Define uma linha em uma tabela HTML.

contém um ou mais elementos <th> ou <td>.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<table>
    <tr>
        <th>Month</th>
        <th>Savings</th>
    </tr>
    <tr>
        <td>January</td>
        <td>$100</td>
    </tr>
    <tr>
        <td>February</td>
        <td>$80</td>
    </tr>
</table>

Configuração Padrão de CSS

                
tr {
    display: table-row;
    vertical-align: inherit;
    border-color: inherit;
}
