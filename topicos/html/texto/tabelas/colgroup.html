<colgroup>

Descrição

Especifica um grupo de uma ou mais colunas em uma tabela para formatação.

Útil para aplicar estilos a colunas inteiras, em vez de repetir os estilos para cada célula, para cada linha.

Notas

a <colgroup> deve ser filha de um elemento <table>, depois de qualquer elemento <caption> e antes de qualquer elemento <thead>, <tbody>, <tfoot> e <tr>.

Atributos

span - Especifica o número de colunas que um grupo de colunas deve abranger.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<table>
    <colgroup>
        <col span="2" style="background-color:red">
        <col style="background-color:yellow">
    </colgroup>
    <tr>
        <th>ISBN</th>
        <th>Title</th>
        <th>Price</th>
    </tr>
    <tr>
        <td>3476896</td>
        <td>My first HTML</td>
        <td>$53</td>
    </tr>
</table>

Configuração Padrão de CSS

                
colgroup {
    display: table-column-group;
}
