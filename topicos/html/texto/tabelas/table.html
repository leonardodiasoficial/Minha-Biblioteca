<table>

Descrição

Define uma tabela HTML.

Uma tabela HTML consiste em um <table> e um ou mais elementos <tr>, <th> e <td>.

O elemento <tr> define uma linha da tabela, o elemento <th> define um cabeçalho da tabela e o elemento <td> define uma célula da tabela.

Uma tabela HTML também pode incluir elementos <caption>, <colgroup>, <thead>, <tfoot> e <tbody>.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<table>
    <tr>
        <th>Month</th>
        <th>Savings</th>
    </tr>
    <tr>
        <td>January</td>
        <td>$100</td>
    </tr>
</table>

Configuração Padrão de CSS

                
table {
    display: table;
    border-collapse: separate;
    border-spacing: 2px;
    border-color: gray;
}
