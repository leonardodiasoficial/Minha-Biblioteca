<dd>

Descrição

Usada para descrever um termo/nome em uma lista de descrição.

Usada em conjunto com <dl> (define uma lista de descrição) e <dt> (define termos/nomes).

Dentro de uma tag <dd> você pode colocar parágrafos, quebras de linha, imagens, links, listas, etc.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<dl>
    <dt>Coffee</dt>
    <dd>Black hot drink</dd>
    <dt>Milk</dt>
    <dd>White cold drink</dd>
</dl>

Configuração Padrão de CSS

                
dd {
    display: block;
    margin-left: 40px;
}
