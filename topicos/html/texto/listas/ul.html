<ul>

Descrição

Define uma lista não ordenada (marcada).

Use a tag junto com a tag <li> para criar listas não ordenadas.

Notas

Use CSS para estilizar listas.

Para listas ordenadas, use a tag <ol>.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<ul>
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
</ul>

Configuração Padrão de CSS

                
ul {
    display: block;
    list-style-type: disc;
    margin-top: 1em;
    margin-bottom: 1 em;
    margin-left: 0;
    margin-right: 0;
    padding-left: 40px;
}
