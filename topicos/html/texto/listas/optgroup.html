<optgroup>

Descrição

Usada para agrupar opções relacionadas em um elemento <select> (lista suspensa).

Se você tiver uma longa lista de opções, os grupos de opções relacionadas serão mais fáceis de manipular para um usuário.

Atributos

disabled - Especifica que um grupo de opções deve ser desabilitado.

label - Especifica um rótulo para um grupo de opções.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<select  name="cars" id="cars">
    <optgroup label="Swedish Cars">
        <option value="volvo">Volvo</option>
        <option value="saab">Saab</option>
    </optgroup>
    <optgroup label="German Cars">
        <option value="mercedes">Mercedes</option>
        <option value="audi">Audi</option>
    </optgroup>
</select>
