<li>

Descrição

Define um item de lista.

Usada dentro de listas ordenadas (<ol>), listas não ordenadas (<ul>) e em listas de menu (<menu>).

Em <ul> e <menu>, os itens da lista geralmente serão exibidos com marcadores.

Em <ol>, os itens da lista geralmente serão exibidos com números ou letras.

Notas

Use CSS para estilizar listas.

Atributos

value - Somente para listas <ol>. Especifica o valor inicial de um item de lista. Os itens da lista a seguir serão incrementados a partir desse número.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<ol>
    <li>Coffee</li>
    <li>Tea</li>
    <li>Milk</li>
</ol>

Configuração Padrão de CSS

                
li {
    display: list-item;
}
