<select>

Descrição

Usado para criar uma lista suspensa.

Usado com mais frequência em um formulário para coletar a entrada do usuário.

O atributo name é necessário para fazer referência aos dados do formulário após o envio do formulário (se você omitir o nameatributo, nenhum dado da lista suspensa será enviado).

O atributo id é necessário para associar a lista suspensa a um rótulo.

As tags <option> dentro do <select> definem as opções disponíveis na lista suspensa.

Notas

Sempre adicione a tag <label> para melhores práticas de acessibilidade!

Atributos

autofocus - Especifica que a lista suspensa deve obter foco automaticamente quando a página for carregada.

disabled - Especifica que uma lista suspensa deve ser desativada.

form - Define a qual formulário a lista suspensa pertence.

multiple - Especifica que várias opções podem ser selecionadas ao mesmo tempo.

name - Define um nome para a lista suspensa.

required - Especifica que o usuário deve selecionar um valor antes de enviar o formulário.

size - Define o número de opções visíveis em uma lista suspensa.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<select name="cars" id="cars">
    <option value="volvo">Volvo</option>
    <option value="saab">Saab</option>
    <option value="mercedes">Mercedes</option>
    <option value="audi">Audi</option>
</select>
