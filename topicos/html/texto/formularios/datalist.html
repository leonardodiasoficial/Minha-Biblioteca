<datalist>

Descrição

Especifica uma lista de opções predefinidas para um elemento <input>.

Usada para fornecer um recurso de "preenchimento automático" para elementos <input>. Os usuários verão uma lista suspensa de opções predefinidas à medida que inserem os dados.

Atributo id do elemento deve ser igual ao atributo list do elemento <input> (isso os une).

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<datalist id="browsers">
    <option value="Edge">
    <option value="Firefox">
</datalist>

Configuração Padrão de CSS

                
datalist {
    display: none;
}

Compatibilidade dos Navegadores

  • Edge = ✔️ (10.0)

  • Firefox = ✔️ (4.0)

  • Safari = ✔️ (12.1)
