<label>

Descrição

Define um rótulo para vários elementos:

<tipo de entrada="caixa de seleção">

<tipo de entrada="cor">

<tipo de entrada="data">

<input type="datetime-local">

<tipo de entrada="e-mail">

<tipo de entrada="arquivo">

<tipo de entrada="mês">

<tipo de entrada="número">

<input type="senha">

<tipo de entrada="rádio">

<tipo de entrada="intervalo">

<input type="pesquisar">

<input type="tel">

<tipo de entrada="texto">

<tipo de entrada="tempo">

<input type="url">

<input type="semana">

<metro>

<progresso>

<selecionar>

<área de texto>

O uso adequado de rótulos com os elementos acima beneficiará:

Usuários de leitores de tela (lerão o rótulo em voz alta, quando o usuário estiver focado no elemento).

Usuários que têm dificuldade de clicar em regiões muito pequenas (como checkboxes) - pois quando um usuário clica no texto dentro do <label>, ele alterna a entrada (isso aumenta a área de acerto).

Notas

O atributo for de <label> deve ser igual ao atributo id do elemento relacionado para uni-los. Um rótulo também pode ser vinculado a um elemento colocando o elemento dentro do >label>.

Atributos

for - Especifica o id do elemento de formulário ao qual o rótulo deve ser vinculado

form - Especifica a qual formulário o rótulo pertence

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<label for="html">HTML</label><br>

Configuração Padrão de CSS

                
label {
    cursor: default;
}
