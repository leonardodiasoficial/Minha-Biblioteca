<textarea>

Descrição

Define um controle de entrada de texto de várias linhas.

Geralmente é usado em um formulário para coletar entradas do usuário, como comentários ou revisões.

Uma área de texto pode conter um número ilimitado de caracteres e o texto é renderizado em uma fonte de largura fixa (geralmente Courier).

O tamanho de uma área de texto é especificado pelos atributos colse rows(ou com CSS).

O name atributo é necessário para fazer referência aos dados do formulário após o envio do formulário (se você omitir o name atributo, nenhum dado da área de texto será enviado).

O idatributo é necessário para associar a área de texto a um rótulo.

Notas

Sempre adicione a tag <label> para melhores práticas de acessibilidade!

Atributos

autofocus - Especifica que uma área de texto deve obter foco automaticamente quando a página for carregada.

cols - Especifica a largura visível de uma área de texto.

dirname - Especifica que a direção do texto da área de texto será enviada.

disabled - Especifica que uma área de texto deve ser desativada.

form - Especifica a qual formulário a área de texto pertence.

maxlength - Especifica o número máximo de caracteres permitidos na área de texto.

name - Especifica um nome para uma área de texto.

placeholder - Especifica uma dica curta que descreve o valor esperado de uma área de texto.

readonly - Especifica que uma área de texto deve ser somente leitura.

required - Especifica que uma área de texto é obrigatória/deve ser preenchida.

rows - Especifica o número visível de linhas em uma área de texto.

wrap - Especifica como o texto em uma área de texto deve ser quebrado quando enviado em um formulário.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<textarea id="w3review" name="w3review" rows="4" cols="50">
    At w3schools.com you.
</textarea>
