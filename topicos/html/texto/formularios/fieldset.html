<fieldset>

Descrição

Usada para agrupar elementos relacionados em um formulário.

Desenha uma caixa ao redor dos elementos relacionados.

Notas

A tag <legend> é usada para definir uma legenda para o <fieldset>.

Atributos

disabled - Especifica que um grupo de elementos de formulário relacionados deve ser desabilitado.

form - Especifica a qual formulário o conjunto de campos pertence.

name - Especifica um nome para o fieldset.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<form action="/action_page.php">
<fieldset>
    <legend>Personalia:</legend>
    <label for="fname">First name:</label>
    <input type="text" id="fname" name="fname"><br><br>
    <label for="lname">Last name:</label>
    <input type="text" id="lname" name="lname"><br><br>
</fieldset>
</form>

Configuração Padrão de CSS

                
fieldset {
    display: block;
    margin-left: 2px;
    margin-right: 2px;
    padding-top: 0.35em;
    padding-bottom: 0.625em;
    padding-left: 0.75em;
    padding-right: 0.75em;
    border: 2px groove (internal value);
}
