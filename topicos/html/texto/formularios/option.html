<option>

Descrição

Define uma opção em uma lista de seleção.

Os elementos vão dentro de um elemento <select> , <optgroup> ou <datalist>.

Notas

A <option> tag pode ser usada sem nenhum atributo, mas geralmente é necessário o atributo value , que indica o que é enviado ao servidor no envio do formulário.

Se você tiver uma longa lista de opções, poderá agrupar as opções relacionadas na tag <optgroup>.

Atributos

disabled - Especifica que uma opção deve ser desativada.

label - Especifica um rótulo mais curto para uma opção.

selected - Especifica que uma opção deve ser pré-selecionada quando a página for carregada.

value - Specifies the value to be sent to a server.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<select id="cars">
    <option value="volvo">Volvo</option>
</select>

Configuração Padrão de CSS

                
Não tem.
