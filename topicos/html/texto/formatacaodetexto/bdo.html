<bdo>

Descrição

Usada para substituir a direção do texto atual.

Atributos

dir - Requeridos. Especifica a direção do texto dentro do elemento <bdo>.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<bdo dir="rtl">
    This text will go right-to-left.
</bdo>

Configuração Padrão de CSS

                
bdo {
    unicode-bidi: bidi-override;
}
