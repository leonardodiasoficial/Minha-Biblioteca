<sub>

Descrição

define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e, às vezes, é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<p>
    This text <sub>subscript</sub> text.
</p>

Configuração Padrão de CSS

                
sub {
    vertical-align: sub;
    font-size: smaller;
}
