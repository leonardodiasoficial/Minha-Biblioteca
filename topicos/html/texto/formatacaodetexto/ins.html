<ins>

Descrição

Define um texto que foi inserido em um documento. Os navegadores geralmente sublinham o texto inserido.

Notas

Observe também a tag <del> para marcar o texto excluído.

Atributos

cite - Especifica uma URL para um documento que explica o motivo pelo qual o texto foi inserido/alterado.

datetime - Especifica a data e hora em que o texto foi inserido/alterado.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<p>
    Favorite color is <del>blue</del> <ins>red</ins>!
</p>

Configuração Padrão de CSS

                
ins {
    text-decoration: underline;
}
