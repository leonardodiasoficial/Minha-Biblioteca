<del>

Descrição

Define o texto que foi excluído de um documento. Os navegadores geralmente traçam uma linha através do texto excluído.

Notas

Observe também a tag <ins> para marcar o texto inserido.

Atributos

cite - Especifica uma URL para um documento que explica o motivo pelo qual o texto foi excluído/alterado

datetime - Especifica a data e a hora em que o texto foi excluído/alterado

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<p>
    Favorite color is <del>blue</del> <ins>red</ins>!
</p>

Configuração Padrão de CSS

                
del {
    text-decoration: line-through;
}
