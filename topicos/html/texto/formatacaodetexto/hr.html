<hr>

Descrição

Define uma quebra temática em uma página HTML (por exemplo, uma mudança de tópico).

Geralmente é exibido como uma régua horizontal usada para separar o conteúdo (ou definir uma alteração) em uma página HTML.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<p>
    HTML is the standard.
</p>

<hr>

<p>
    CSS is a language.
</p>

Configuração Padrão de CSS

                
hr {
    display: block;
    margin-top: 0.5em;
    margin-bottom: 0.5em;
    margin-left: auto;
    margin-right: auto;
    border-style: inset;
    border-width: 1px;
}
