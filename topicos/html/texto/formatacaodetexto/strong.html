<strong>

Descrição

Usada para definir o texto com forte importância. O conteúdo interno geralmente é exibido em negrito.

Notas

Use a tag <b> para especificar o texto em negrito sem nenhuma importância extra!

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<strong>
    This text is important!
</strong>

Configuração Padrão de CSS

                
strong {
    font-weight: bold;
}
