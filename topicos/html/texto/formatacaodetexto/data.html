<data>

Descrição

Usada para adicionar uma tradução legível por máquina de um determinado conteúdo.

Esse elemento fornece um valor legível por máquina para processadores de dados e um valor legível por humanos para renderização em um navegador.

Notas

Se o conteúdo for relacionado a data ou hora, use o elemento <time> em vez disso.

Atributos

value - Especifica a tradução legível por máquina do conteúdo do elemento.

Atributos Globais - A tag suporta os atributos globais em HTML.

Exemplo

                
<li>
    <data value="21053">Cherry Tomato</data>
</li>
