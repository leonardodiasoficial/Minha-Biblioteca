<output>

Descrição

Usada para representar o resultado de um cálculo (como um realizado por um script).

Atributos

for - Especifica a relação entre o resultado do cálculo e os elementos usados ​​no cálculo

form - Especifica a qual formulário o elemento de saída pertence

name - Especifica um nome para o elemento de saída

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<form oninput="x.value=parseInt(a.value)+parseInt(b.value)">
    <input type="range" id="a" value="50">
    +<input type="number" id="b" value="25">
    =<output name="x" for="a b"></output>
</form>

Configuração Padrão de CSS

                
output {
    display: inline;
}
