<q>

Descrição

Define uma citação curta.

Os navegadores normalmente inserem aspas ao redor da citação.

Notas

Use <blockquote> para citações longas.

Atributos

cite - Especifica o URL de origem da citação

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<p>
    WWF's goal is to:
    <q>Build a future.</q>
    We hope they succeed.
</p>

Configuração Padrão de CSS

                
q {
    display: inline;
}
    
q:before {
    content: open-quote;
}
    
q:after {
    content: close-quote;
}
