<s>

Descrição

Especifica o texto que não é mais correto, preciso ou relevante. O texto será exibido com uma linha através dele.

não deve ser usada para definir texto deletado em um documento, use a tag <del> para isso.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<p>
    <s>Only 50 tickets left!</s>
</p>

Configuração Padrão de CSS

                
s {
    text-decoration: line-through;
}
