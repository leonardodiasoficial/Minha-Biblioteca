<time>

Descrição

Define uma hora específica (ou datetime).

Atributo desse elemento é usado para traduzir a hora em um formato legível por máquina para que os navegadores possam adicionar lembretes de data por meio do calendário do usuário e os mecanismos de pesquisa possam produzir resultados de pesquisa mais inteligentes.

Atributos

datetime - Representa um formato legível por máquina do elemento <time>

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<p>
    I have a date on <time datetime="2008-02-14 20:00">Valentines day</time>.
</p>

Configuração Padrão de CSS

                
Não tem.
