<i>

Descrição

Define uma parte do texto em uma voz ou humor alternativo. O conteúdo interno geralmente é exibido em itálico.

Frequentemente usada para indicar um termo técnico, uma frase de outro idioma, um pensamento, um nome de navio, etc.

Use o elemento somente quando não houver um elemento semântico mais apropriado, como:

- <em> (texto em itálico)

- <strong> (texto em negrito)

- <mark> (texto grifado)

- <cite> (o título de um trabalho)

- <dfn> (um termo de definição)

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<p>
    <i>Lorem ipsum</i> is the most popular.
</p>

Configuração Padrão de CSS

                
i {
    font-style: italic;
}
