<var>

Descrição

Usada para definir uma variável na programação ou em uma expressão matemática. O conteúdo interno geralmente é exibido em itálico.

Notas

Esta tag não está obsoleta. No entanto, é possível obter um efeito mais rico usando CSS.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<p>
    The triangle: 1/2 x <var>b</var> x <var>h</var>.
</p>

Configuração Padrão de CSS

                
var {
    font-style: italic;
}
