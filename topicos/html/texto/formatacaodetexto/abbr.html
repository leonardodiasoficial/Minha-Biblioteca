<abbr>

Descrição

define uma abreviação ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

Notas

Use o atributo global para mostrar a descrição da abreviação/acrônimo ao passar o mouse sobre o elemento.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.

Configuração Padrão de CSS

                
abbr {
    display: inline;
}
