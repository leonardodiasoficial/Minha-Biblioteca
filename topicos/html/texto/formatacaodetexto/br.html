<br>

Descrição

Insere uma única quebra de linha.

Útil para escrever endereços ou poemas.

Uma tag vazia, o que significa que não tem tag final.

Notas

use a <br> para inserir quebras de linha, não para adicionar espaço entre os parágrafos.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<p>
    To force<br> line breaks<br> in a text,<br> use the br<br> element.
</p>
