<blockquote>

Descrição

Especifica uma seção citada de outra fonte

Os navegadores geralmente indentam <blockquote>.

Notas

Use <q> para citações inline (curtas).

Atributos

cite - Especifica a fonte da citação

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<blockquote cite="http://www.worldwildlife.org/who/index.html">
    For 50 years, WWF has been protecting the future of nature.
</blockquote>

Configuração Padrão de CSS

                
blockquote {
    display: block;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 40px;
    margin-right: 40px;
}
