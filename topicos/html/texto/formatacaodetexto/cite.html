<cite>

Descrição

Define o título de uma obra criativa (por exemplo, um livro, um poema, uma música, um filme, uma pintura, uma escultura, etc.).

Notas

O nome de uma pessoa não é o título de uma obra.

O texto no <cite> geralmente é renderizado em itálico.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<p>
    <cite>The Scream</cite> by Edward Munch.
</p>

Configuração Padrão de CSS

                
cite {
    font-style: italic;
}
