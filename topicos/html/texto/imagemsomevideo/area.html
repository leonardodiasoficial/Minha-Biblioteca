<area>

Descrição

Define uma área dentro de um mapa de imagem (um mapa de imagem é uma imagem com áreas clicáveis).

Os elementos estão sempre aninhados dentro de uma <map>.

Notas

O usemapatributo in <img> está associado ao atributo <map> do elemento name, e cria uma relação entre a imagem e o mapa.

Atributos

alt - Especifica um texto alternativo para a área. Obrigatório se o atributo href estiver presente.

coords - Especifica as coordenadas da área.

download - Especifica que o destino será baixado quando um usuário clicar no hiperlink.

href - Especifica o destino do hiperlink para a área.

hreflang - Especifica o idioma do URL de destino.

media - Especifica para qual mídia/dispositivo o URL de destino é otimizado.

referrerpolicy - Especifica quais informações do referenciador enviar com o link.

rel - Especifica a relação entre o documento atual e o URL de destino.

shape - Especifica a forma da área.

target - Especifica onde abrir o URL de destino.

type - Especifica o tipo de mídia do URL de destino.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<map name="workmap">
    <area shape="rect" coords="34,44,270,350" alt="Computer" href="computer.htm">
    <area shape="rect" coords="290,172,333,250" alt="Phone" href="phone.htm">
    <area shape="circle" coords="337,300,44" alt="Cup of coffee" href="coffee.htm">
</map>

Configuração Padrão de CSS

                
area {
    display: none;
}
