<figure>

Descrição

Especifica o conteúdo independente, como ilustrações, diagramas, fotos, listagens de código, etc.

Enquanto o conteúdo do <figure> está relacionado ao fluxo principal, sua posição é independente do fluxo principal e, se removido, não deve afetar o fluxo do documento.

Notas

O elemento <figcaption> é usado para adicionar uma legenda para o <figure>.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<figure>
    <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
    <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>
</figure>

Configuração Padrão de CSS

                
figure {
    display: block;
    margin-top: 1em;
    margin-bottom: 1em;
    margin-left: 40px;
    margin-right: 40px;
}
