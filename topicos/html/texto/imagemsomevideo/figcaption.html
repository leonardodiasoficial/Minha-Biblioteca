<figcaption>

Descrição

Define uma legenda para um elemento <figure>.

O <figcaption> pode ser colocado como o primeiro ou último filho do elemento <figure>.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos de Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<figure>
    <img src="pic_trulli.jpg" alt="Trulli" style="width:100%">
    <figcaption>Fig.1 - Trulli, Puglia, Italy.</figcaption>
</figure>

Configuração Padrão de CSS

                
figcaption {
    display: block;
}
