<html>

Descrição

Representa a raiz de um documento HTML.
É o recipiente para todos os outros elementos HMTL (exceto para a tag <!DOCTYPE>).

Notas

Você deve sempre incluir o atributo lang dentro da tag, para declarar o idioma da página na Web. Isso se destina a ajudar os motores de busca e navegadores.

Atributos

Atributos Globais - A tag suporta os atributos globais em HTML.

Exemplo

                
<html lang="en">
<head>
    <title>Title of the document</title>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

Configuração Padrão de CSS

                
html {
    display: block;
    }
    
html:focus {
    outline: none;
    }
