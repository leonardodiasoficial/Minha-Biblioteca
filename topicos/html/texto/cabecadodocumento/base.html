<base>

Descrição

Especifica a URL base e/ou destino para todas as URLs relativas em um documento.

Deve ter um href ou um atributo de destino presente, ou ambos.

Só pode haver um único <base> em um documento e ele deve estar dentro da tag <head>.

Atributos

href - Especifica o URL base de todos os URLs relativos na página.

target - Especifica o destino padrão para todos os hiperlinks e formulários na página.

Atributos Globais - A tag suporta os atributos globais em HTML.

Exemplo

                
<base href="https://www.w3schools.com/" target="_blank">

Configuração Padrão de CSS

                
Não suporta a Configuração do CSS
