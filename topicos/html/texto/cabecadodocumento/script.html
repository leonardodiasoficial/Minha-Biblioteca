<script>

Descrição

Usada para incorporar um script do lado do cliente (JavaScript).

Contém instruções de script ou aponta para um arquivo de script externo por meio do atributo src.

Usos comuns para JavaScript são manipulação de imagem, validação de formulário e alterações dinâmicas de conteúdo.

Notas

Observe também o elemento <noscript> para usuários que desabilitaram scripts em seus navegadores ou têm um navegador que não oferece suporte a scripts do lado do cliente.

Atributos

async - Especifica que o script é baixado em paralelo à análise da página e executado assim que estiver disponível (antes da conclusão da análise) (somente para scripts externos).

crossorigin - Define o modo da solicitação para uma solicitação HTTP CORS.

defer - Especifica que o script é baixado em paralelo à análise da página e executado após a conclusão da análise da página (somente para scripts externos).

integrity - Permite que um navegador verifique o script obtido para garantir que o código nunca seja carregado se a fonte tiver sido manipulada.

nomodule - Especifica que o script não deve ser executado em navegadores com suporte a módulos ES2015.

referrerpolicy - Especifica quais informações do referenciador enviar ao buscar um script.

src - Especifica a URL de um arquivo de script externo.

type - Especifica o tipo de mídia do script.

Atributos Globais - A tag suporta os atributos globais em HTML.

Exemplo

                
<script>
    document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>

Configuração Padrão de CSS

                
script {
    display: none;
    }
