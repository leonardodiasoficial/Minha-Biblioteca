<style>

Descrição

Usada para definir informações de estilo (CSS) para um documento.

Dentro do <style>, você especifica como os elementos HTML devem ser renderizados em um navegador.

Notas

Quando um navegador lê uma folha de estilo, ele formata o documento HTML de acordo com as informações da folha de estilo. Caso algumas propriedades tenham sido definidas para o mesmo seletor (elemento) em diferentes folhas de estilo, será utilizado o valor da última folha de estilo lida!

Para vincular a uma folha de estilo externa, use a tag <link>.

Atributos

media - Especifica para qual mídia/dispositivo o recurso de mídia é otimizado.

type - Especifica o tipo de mídia da tag <style>.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos do Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<style>
    h1 {color:red;}
    p {color:blue;}
</style>

Configuração Padrão de CSS

                
style {
    display: none;
    }
