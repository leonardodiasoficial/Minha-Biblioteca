<head>

Descrição

É um contêiner para metadados (dados sobre dados) e é colocado entre as tags <html> e <body>.
Metadados são dados sobre o documento HTML. Os metadados não são exibidos.
Os metadados geralmente definem o título do documento, conjunto de caracteres, estilos, scripts e outras metainformações.
Os seguintes elementos podem ir dentro do <head>:
- <title> (exibido em todo documento HTML)
- <style>
- <base>
- <link>
- <meta>
- <script>
- <noscript>

Atributos

Atributos Globais - A tag suporta atributos globais em HTML.

Exemplo

                
<head>
    <title>Title of the document</title>
</head>

Configuração Padrão de CSS

                
head {
    display: none;
}
