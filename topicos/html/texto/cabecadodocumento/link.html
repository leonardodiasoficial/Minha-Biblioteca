<link>

Descrição

Define o relacionamento entre o documento atual e um recurso externo.

Usada com mais frequência para vincular a folhas de estilo externas ou para adicionar um favicon ao seu site.

É um elemento vazio, contém apenas atributos.

Atributos

crossorigin - Especifica como o elemento lida com solicitações de origem cruzada.

href - Especifica a localização do documento vinculado.

hreflang - Especifica o idioma do texto no documento vinculado.

media - Especifica em qual dispositivo o documento vinculado será exibido.

referrerpolicy - Especifica qual referenciador usar ao buscar o recurso.

rel - Requeridos. Especifica a relação entre o documento atual e o documento vinculado.

sizes - Especifica o tamanho do recurso vinculado. Somente para rel="icon".

title - Define uma folha de estilo preferida ou alternativa.

type - Especifica o tipo de mídia do documento vinculado.

Atributos Globais - A tag suporta os atributos globais em HTML.

Atributos do Evento - A tag oferece suporte aos atributos de evento em HTML.

Exemplo

                
<link rel="stylesheet" href="styles.css">

Configuração Padrão de CSS

                
link {
    display: none;
    }
