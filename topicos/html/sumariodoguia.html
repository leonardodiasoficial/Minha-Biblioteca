Tags e Atributos

  • html - Esta tag especifica que a página foi escrita em HTML. Ela aparece na primeira linha da página. É utilizada principalmente para mostrar que a página usa HTML5. Esta tag pode ser considerada como uma tag pai para qualquer outra tag na página.

  • head - Esta tag é utilizada para especificar dados meta sobre a página. Ela inclui o nome da página, suas dependências (JS e scripts CSS), uso da fonte, etc.

  • title - Esta tag contém o título/nome da página. Você pode ver isso na barra de navegação do navegador para qualquer página aberta. Mecanismos de busca utilizam esta tag para extrair o assunto da página, o que é bastante conveniente para ranqueamento.

  • body - Tudo que o usuário vê é escrito dentro desta tag, que contém todos os conteúdos da página.
