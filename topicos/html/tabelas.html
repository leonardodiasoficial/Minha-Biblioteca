Tags e Atributos

  • table - Marca uma tabela em uma página.

  • caption - Uma descrição da tabela é colocada dentro desta tag.

  • thead - Especifica informações referente a colunas específicas da tabela.

  • tbody - O conteúdo de uma tabela, onde os dados são mantidos.

  • tfoot - Determina o rodapé de uma tabela.

  • tr - Denota uma única fileira em uma tabela.

  • th - O valor de um cabeçalho de uma coluna de tabela.

  • td - um único campo de uma tabela. Contém o valor/dado atual.

  • colgroup - Usada para agrupar colunas juntas.

  • col - Denota uma coluna dentro de uma tabela.
