Tags e Atributos

  • a href="" - Tag âncora. Primeiramente utilizada para inclusão de hyperlinks.

  • a href="mailto:" - Tag dedicada para o envio de emails.

  • a href="tel://###-###" - Tag âncora para menção de números de contato. Como os números são aplicáveis, é particularmente útil para usuários de celular.

  • a name="name" - Esta tag pode ser utilizada para navegação rápida para outra parte da página.

  • a href="#name" - Uma variação da tag acima, sendo utilizada somente para navegação de uma seção div.
