Tags e Atributos

  • form - A tag pai de um formulário HTML

  • fieldset - Identifica o grupo de todos os campos no formulário.

  • label - É utilizado para marcar um campo no formulário.

  • legend - Opera como legenda para o elemento fieldset.

  • input - Esta tag é utilizada para receber informações (input) do usuário. O tipo de informação é determinado por um número de atributos.

  • select - Usado para criar uma lista suspensa. Usado com mais frequência em um formulário para coletar a entrada do usuário.

  • datalist - Especifica uma lista de opções predefinidas para um elemento input.

  • textarea - Define um controle de entrada de texto de várias linhas. Usado no formulário para coletar entradas do usuário, como comentários ou revisões.
