Tags e Atributos

  • form - A tag pai de um formulário HTML

  • fieldset - Identifica o grupo de todos os campos no formulário.

  • label - É utilizado para marcar um campo no formulário.

  • legend - Opera como legenda para o elemento fieldset.

  • input - Esta tag é utilizada para receber informações (input) do usuário. O tipo de informação é determinado por um número de atributos.
