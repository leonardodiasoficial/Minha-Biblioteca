Tags e Atributos

  • header - Especifica o cabeçalho da página. Também pode ser utilizado para objetos dentro da página.

  • footer - Especifica o rodapé da página. Também pode ser utilizada para objetos dentro da página.

  • main - Marca o conteúdo principal da página.

  • article - Denota um artigo.

  • aside - Denota o conteúdo mostrado em um canto da página.

  • section - Especifica uma seção específica da página.

  • details - Usado para informações adicionais. O usuário tem a opção de visualizar ou esconder isso.

  • summary - Usado como cabeçalho para a tag acima. Sempre visível para o usuário.

  • dialog - Usado para criar uma caixa de diálogo.

  • figure - Uma tag reservada para figuras (diagrama, gráficos) no HTML5.

  • figcaption - Uma descrição de uma figura colocada dentro dessa.

  • mark - Usado para destacar uma parte específica do texto.

  • nav - Links de navegação para o usuário em uma página.

  • menuitem - Um ítem específico de uma lista ou menu.

  • meter - Mediação de dados dentro de um parâmetro definido.

  • progress - Tipicamente usado como barra de progresso, usado para medir o progresso.

  • rp - Esta tag é para mostrar o texto em navegadores que não possuem suporte para ruby.

  • rt - Mostra detalhes de caracteres do leste asiático.

  • ruby - Descreve uma anotação Ruby para tipografia do leste asiático.

  • time - Tag para formatação de dia e hora.

  • wbr - Uma quebra de linha dentro do conteúdo.
