Tags

  • ol - Usada para lista ordenada ou numerada de itens.

  • ul - Usada para itens desordenadas.

  • li - Item individual como parte da lista.

  • dl - Usada para listas de itens com definições.

  • dt - A definição de um único termo em linha com conteúdo.

  • dd - Descrição do termo definido.

  • bdi - Isola uma parte do texto que pode estar formada em uma direção diferente de outro texto fora dela.

  • optgroup - Usada para agrupar opções relacionadas em um elemento select (lista suspensa).
