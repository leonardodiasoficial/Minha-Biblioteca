Tags

  • body - Define o corpo da página

  • header - Define o cabeçalho de um documento ou seção declarado no HTML.

  • footer - Define o rodapé do site ou de uma seção.

  • main - Define o conteúdo principal da página.

  • article - Denota um artigo.

  • aside - Define um conteúdo de sentido independente do restante do documento HTML.

  • section - Define seções/tópicos de um documento.

  • details - Usado para informações adicionais. O usuário tem a opção de visualizar ou esconder isso.

  • summary - Usado como cabeçalho para a tag acima. Sempre visível para o usuário.

  • dialog - Usado para criar uma caixa de diálogo.

  • nav - Define um grupo de links de navegação.

  • div - Define uma divisão ou seção em um documento HTML.

  • menu - Define uma lista de comandos. Usado para criar menus de contexto, barras de ferramentas, listar controles de formulários e comandos.

  • ruby - Descreve uma anotação Ruby para tipografia do leste asiático.

  • template - Um mecanismo para encapsular um conteúdo do lado do cliente que não é renderizado quando a página é carregada, mas que pode ser instanciado posteriormente em tempo de execução usando JavaScript.
