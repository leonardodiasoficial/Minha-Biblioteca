Tags

  • p - Define um parágrafo.

  • h1...h6 - Define um cabeçalho (Título).

  • b - Define um texto em negrito sem semântica.

  • em - Usado para dar ênfase em um texto, enfatizando.

  • i - Usado para deixar o texto em itálico, mas não enfatiza como a tag acima.

  • cite - Usado para citar o autor de uma frase.

  • strong - Define um texto em negrito com semântica.

  • del - Texto pré-formatado "monoespaçado" apresentado com espaço em branco dentro do elemento intacto.

  • ins - Denota um texto que foi inserido na página.

  • blockquote - Define uma seção que é uma citação de outra fonte.

  • q - Define uma citação.

  • abbr - Define uma abreviação ou acronimo.

  • address - Usado para especificar as informações de contato do autor.

  • dfn - Usada para definições.

  • code - Define um pedaço de código de programação.

  • sub - Usada para escrita de um subscript (um texto de fonte menor logo abaixo da fonte principal).

  • sup - Similar com a tag anterior, mas para superscript (texto acima).

  • small - Reduz o tamanho do texto. Se refere à informações redundantes ou inválidas.

  • bdo - Usado para substituir a direcionalidade atual do texto.

  • data - Usada para adicionar uma tradução legível por máquina de um determinado conteúdo.

  • s - Especifica o texto que não é mais correto, preciso ou relevante. O texto será exibido com uma linha no meio dele.

  • samp - Usada para definir a saída de amostra de um programa de computador. O conteúdo interno é exibido na fonte monoespaçada padrão do navegador.

  • u - Marca um texto não articulado e com estilo diferente do texto normal, como palavras com erros ortográficos ou nomes próprios em texto chinês. É exibido com um sublinhado.

  • output - Usada para representar o resultado de um cálculo.

  • var - Usada para definir uma variável na programação ou em uma expressão matemática. É exibido em itálico.

  • kdb - Usada para definir a entrada do teclado. É eixibido na fonte monoespaçada padrão do navegador.

  • rt - Mostra os detalhes de caracteres do leste asiático.

  • rp - Mostrar texto em navegadores que não possuem suporte para Ruby.

  • time - Usada para formatação de dia e hora.

  • mark - Usado para destacar uma parte especídica do texto.

  • span - Um contêiner embutido usado para marcar uma parte de um texto ou uma parte de um documento.

  • br - Define uma quebra de linha.

  • button - Define um botão clicável

  • pre - Define um texto pré-formatado.

  • hr - Define uma quebra temática entre elementos de nível de parágrafo.
