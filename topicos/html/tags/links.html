Tags

  • a href="" - Usada para inclusão de hyperlinks.

  • a href="mailto:" - Usada para o envio de emails.

  • a href="tel://###-###" - Usada para menção de números de contato.

  • a href="name" - Usada para navegação rápida para outra parte da página.

  • a href="#name" - Uma variação da tag acima, sendo utilizada somente para navegação de uma seção div.
