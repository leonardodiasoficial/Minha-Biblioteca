Tags

  • img - Mostrar imagens em uma página.

  • video - Usada para incorporar conteúdo de vídeo em um documento.

  • audio - Usada para incorporar conteúdo de som em um documento.

  • picture - Oferece aos desenvolvedores mais flexibilidade na especificação de recursos de imagem.

  • source - Usada para especificar vários recursos de mídia para elementos de mídia.

  • figure - Uma tag reservada para figuras (diagramas, gráficos).

  • figcaption - Uma descrição de uma figura colocada dentro dessa.

  • map - Denota uma imagem interativa (clicável).

  • area - Especifica a área de imagem do mapa

  • track - Especifica faixas de texto para elementos de audio ou video.

  • svg - Define um container para gráficos svg.
